###  **Patient Health Risk Analysis** 
Healthcare data analysis project focused on patient risk classification and health metrics analysis.

#### Features:
- **Data Processing**: Cleaning and preprocessing patient health data
- **Risk Classification**: Automated risk categorization based on medical metrics (blood pressure, cholesterol, etc.)
- **Statistical Analysis**: Comprehensive statistical summaries and insights
- **Data Visualization**: Visual representation of health trends and risk distribution

#### Technologies Used:
- Python 3
- pandas - Data manipulation and analysis
- NumPy - Numerical computing
- Matplotlib - Data visualization
- Seaborn - Statistical graphics

#### Key Metrics:
- Patient demographics analysis
- Blood pressure and cholesterol monitoring
- Risk level classification (Low, Medium, High)
- Health trend analysis
